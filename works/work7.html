Project 7: Santaâ€™s Message Hub

Overview

This project is a holiday-themed website that allows users, particularly children, to record and send videos to Santa Claus and receive a personalized response. Developed as a school project, the website features a festive and engaging design to enhance the holiday experience. The interface is intuitive, guiding users through a simple step-by-step process to send their videos. The goal was to create an interactive and visually appealing website that captures the joy and excitement of Christmas while ensuring smooth functionality and ease of use.

Technologies Used

  • Adobe XD: Used for designing the entire layout, including UI elements, backgrounds, and icons.
  • Adobe Illustrator custom illustrations: All design elements were drawn manually except for Christmas card images.
  • Responsive Web Design: Ensured compatibility across different screen sizes and devices.

Challenges

One of the biggest challenges was designing and implementing a fully custom UI with hand-drawn illustrations. Ensuring that all elements looked cohesive and visually appealing while maintaining proper responsiveness required extensive design iteration and testing. Another challenge was optimizing the user journey, making sure each step in the video submission process was clear, simple, and engaging for children. Additionally, integrating interactive elements while keeping the site lightweight and fast was a key focus.

Key Features

  • Step-by-Step Guide: Clearly instructs users on how to record and submit their videos to Santa.
  • Contact Form: Users can submit inquiries or get help with the submission process.
  • User-Friendly Navigation: Simple and intuitive layout, ensuring a smooth user experience.
  • Festive Design: A holiday-inspired interface with custom illustrations, creating an immersive Christmas experience.
