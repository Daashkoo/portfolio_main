Project 10: Style Guide & UI Sketch

Overview

A style guide and UI sketch for an interior design website, focusing on a vibrant aesthetic, accessibility, and intuitive navigation. Designed entirely using Adobe Photoshop, Illustrator, and XD, with all elements hand-drawn.

Technologies Used

  • Adobe Photoshop, Illustrator, XD: Designed all UI elements, typography, icons, and layout. Every component was hand-drawn and customized for the project.

Challenges

Ensuring readability, consistency, and cohesion while using a vibrant palette was key. To overcome contrast issues, I carefully paired colors, tested legibility, and used bold typography for clarity.

Key Features

  • Color Palette & Typography: A carefully selected mix of pastel and bold tones, paired with legible and stylish fonts.
  • Custom social media, navigation, and listing icons to improve user interaction.
  • Helper Features: Breadcrumbs, pagination, pop-ups, mini-navigation.
  • Interactive Elements: Buttons, forms, checkboxes, progress bars, alerts.
  • Accessibility Focus: High contrast, clear hover states, structured navigation.
