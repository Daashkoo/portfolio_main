Project 9: Global Warming Awareness Website

Overview

This website aims to raise awareness about global warming through educational content and interactive features. Designed for a school project, it provides an engaging and easy-to-use platform to inform users of all ages about climate change.

Technologies Used

  • HTML & CSS: Structures the layout and provides styling for a visually cohesive design.
  • Modern CSS Techniques: Employed Flexbox, Grid, and animations for design.
  • Responsive Web Design: Ensured compatibility on all devices.

Challenges

Balancing serious content with engaging visuals while ensuring the site was responsive and easy to navigate was challenging. Presenting complex climate data in a simple and digestible way also required careful planning.

Key Features

  • Interactive Graphs and Data: Displayed real-time data on global temperatures, CO2 emissions, and other environmental statistics to help users visualize the effects of climate change.
  • Educational Resources: Easy-to-read content on climate change.
  • Call-to-Action: Encouraged users to take actions to help reduce global warming, such as reducing waste, conserving energy, and supporting green initiatives.
  • Environmentally-Themed Design: Nature-inspired visuals.
  • Interactive Quiz: Test knowledge on global warming.
