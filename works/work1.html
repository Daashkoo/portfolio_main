Project 1: Essential Oils E-Commerce Website

Overview

This project is an E-commerce website developed to sell essential oils online, offering a user-friendly interface for browsing, purchasing, and learning about various essential oils. The website focuses on product descriptions, customer reviews, and easy navigation to create a seamless shopping experience. It includes features like a discount offer for new customers, integration with a shopping cart, and a newsletter subscription for ongoing engagement. The goal was to develop an interactive and visually appealing website that caters to health-conscious consumers interested in essential oils and their benefits..

Technologies Used

  • HTML, CSS, JavaScript
  • React, Node.js
  • MongoDB
