Project 1: Essential Oils E-Commerce Website

Overview

This project is an E-commerce website developed to sell essential oils online, offering a user-friendly interface for browsing, purchasing, and learning about various essential oils. The website focuses on product descriptions, customer reviews, and easy navigation to create a seamless shopping experience. It includes features like a discount offer for new customers, integration with a shopping cart, and a newsletter subscription for ongoing engagement. The goal was to develop an interactive and visually appealing website that caters to health-conscious consumers interested in essential oils and their benefits..

Technologies Used

  • HTML, CSS, JavaScript: For building the structure, design, and interactivity of the site.
  • React, Node.js
  • Mailchimp Integration: To collect email addresses for the newsletter subscription and send out promotional material.

Challenges

One of the key challenges in this project was Mobile Optimization: Ensuring the website is fully responsive and works well on different screen sizes, from mobile phones to tablets and desktops. To overcome this, I implemented a mobile-first approach using CSS media queries to create responsive layouts. I ensured that images were flexible, navigation was optimized for touch devices with a hamburger menu, and fonts were scalable. Additionally, I tested the website across various devices to ensure a seamless user experience, optimizing performance by compressing images and minimizing code.

Key Features

  • Product Listings: Displaying various essential oils with detailed descriptions, prices, and images, allowing users to easily explore products.
  • Customer Reviews: Users can leave reviews on each product to boost trust and reliability.
  • Responsive Design: The website is mobile-friendly, ensuring it works seamlessly across all devices.
  • Visual Design: Clean layout with a minimalist approach and consistent branding to ensure ease of use and a pleasant user experience.
  • Product Filtering: Search and filter products based on categories like "Lavender Oil," "Peppermint Oil," etc.
